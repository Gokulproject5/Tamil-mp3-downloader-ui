How to run this app?

If you are developer, enable this

Developer Mode

Build & run on your own!

1. Fork this and clone your project

2. Open your terminal and run

git clone https://github.com/anburocky3/tamil-mp3-downloader

Replace "anburodky3" with your github username.

3. Run python script like this:

pip install -r requirements.txt

python main.py

Make sure you have Python 3.10+ installed.
